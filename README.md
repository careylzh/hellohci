# Tangible User Interfaces: An HCI Ancestry

A dependency-free static timeline for GitHub Pages.

## Preview

```bash
python3 -m http.server 8000 --directory hci-history
```

Then open <http://localhost:8000/>.

## Deploy

The intended public URL is <https://careylzh.github.io/hellohci/>.

The folder is static and can be published as the root of the `careylzh/hellohci` GitHub Pages project.
