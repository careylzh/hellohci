# HCI Paradigms Around Tangible Bits

## Goal

Use Codex CLI to traverse each concept, build understanding, and produce concise study notes.

| Concept | Key work | Big idea | Understanding built? |
|---|---|---|---|

## Codex CLI Instructions

For each row:

1. Find the canonical paper / source.
2. Read or skim enough to understand the concept.
3. Write 3–5 bullet notes explaining the concept.
4. Explain how it relates to **Tangible Bits**.
5. Mark `Understanding built?` as `Yes` only after notes are completed.
6. Save all notes into a clean Markdown literature-review scaffold.
